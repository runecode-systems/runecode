# Security Policy

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Instead, use GitHub Security Advisories for private reporting:

1. Go to the repository page on GitHub.
2. Click the "Security" tab.
3. Click "Report a vulnerability".

If you are unable to use GitHub Security Advisories, open an issue asking for a
private reporting channel.

## Supported Versions

RuneCode is currently pre-release. Security fixes will be handled on a
best-effort basis.
