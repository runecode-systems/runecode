## Resolution Notes
This feature defines how canonical verification evidence is preserved, packaged, disclosed, and independently checked.

The selected standards keep the work aligned with:

- schema-governed portable object formats
- local-first authoritative evidence persistence and later export
- trusted runtime, approval, and audit evidence binding inside exported verification views
- deterministic check-only verification behavior rather than mutable export shortcuts
